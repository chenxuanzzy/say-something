2021 八月雜記

9月 4 2021 雜記 14 分鐘 讀完 (約 2032 字)

隨著疫情的趨緩，原本居家辦公的我在七月中的時候開始回公司工作了，總覺得時間過好快，感覺五月疫情剛開始很嚴重的時候，還沒搞清楚狀況就居家辦公到七月，八月整整開始都在公司上班（當然一整天也都戴口罩 QQ）

總結一下八月分都做了些什麼

股票課

額外進修的股票課從四月一路上到八月，高階二班沒有繼續上，剛好在八月結束課程到一個段落，覺得有喘口氣的感覺，畢竟從四月到八月每個假日早上三小時還要每周定時繳交 2-5 份功課，壓力不算大但是多少都有點擔心考試沒過，但是也是很順利的畢業了。這幾個月在股票上學習到很多新知識以外，個人的人格特質也被股票老師圈了出來，像是不細心、心煩氣躁等等這些都是我需要改的問題，除了學股票也順便改變自己（？）

工作

在工作上，六月時公司有位 PM 離職了，新來的 PM 還在熟悉狀況，所以七月好像也沒有什麼特別事情要忙，我開始上線上課程，精進自己的程式能力。上了約兩個月多後就沒有繼續購買課程，發現自己會主動開始找資源學習，系統化的學習和自主的練習等等這些習慣，在課程中已經學我得差不多了。不過後續有想要寫後端程式的話我會想要持續購買把它上完。而課程學習的東西確實有應用在工作上，在解 Bug 的時候不管是找資源或是詢問同事上，都有明顯的知道程式是在做什麼，最有趣的是，有一天看程式有一種豁然開朗的感覺 XD，我覺得這些都是課程和練習帶給我的反饋！未來也會持續往這個方向前進，不斷的增進自我。

同事

在疫情趨緩的時候有位新同事到來，叫 Alvin，講話超問柔，也很有耐心！詢問他問題時心裡也會跟著平靜下來，我發現我喜歡講話溫柔的人，所以當我跟他相處的時候我也盡量學習，講話溫柔、小聲且有禮貌。Alvin 在操作網頁 CSS 的功力非常熟練，我之前有跟他說覺得自己寫的 CSS 都不乾淨，他說要依照畫面功能去寫，很多時候不是硬寫就可以解決問題的。而在工作上跟我說了一些觀念跟做法，是位值得學習的同事。

之前跟他聊天有聊到他也是最近才學框架那些，我問說你看文件都不會覺得很煩嗎，我有時候真的很懶得看文件就直接動手，他說他以前也是這樣，後來才發現文件上有太多東西可以避免踩雷，所以他現在都先看文件。他在學習新框架的時候也覺得很痛苦，果然每個人在學習的路上都是痛苦的。

健身

月中旬健身房開始控管人數的開放，我的健身教練找我回去量　 inbody，體重上並沒有因為疫情而放肆，但是肌肉掉了兩公斤，教練推測說可能是我碳水吃太少又只有有氧（之前七月份有一週固定幾天去外面慢跑四五公里，都有戴口罩），回歸健身房之後教練把我的營養素暫時做新的調整，我也結束了斷食減重的方式，變成每天計算好營養素搭配運動，首要目標仍舊是減脂（萬年不變），訓練上的重量跟之前做的都差蠻多的，不過我相信會慢慢變得強壯！我選擇相信自己。

長笛

長笛課也持續進行中，不知不覺在兩個月就滿學一年了，還是個初階者，但是長笛老師有說我跟以往很不同了，現階段要開始在意吹奏的聲音好聽與否（以前只在乎有沒有吹出聲音），現在一天至少會花個三十分鐘練習，有時候吹得不錯會得到師丈的讚許哈哈！

另外老師有詢問我是否有想要購買自己的長笛（目前是跟老師借），因為會有吹習慣以及熟悉的問題，所以老師認為我真的有興趣且願意繼續學下去的話可以購買好一些的長笛，最近也有在網路上看心得分享做了功課，打算明年買一把自己喜歡的中階款長笛（要好好存錢才行><），長笛上有個小轉變是吹長笛的心態，原本是「想要當街頭藝人而去吹奏長笛」轉變成「我想要吹出好聽的音色給自己聽」，是為了自己而吹奏，不是為了別人。

總結

九月開始做了新的生活調整，早上會早起一點看一點英文或是書，之前一直吵說要考多益，覺得也是時候開實踐這些事了（也希望以後能跟姐姐的男友交流順暢，動力之一），也另外記錄了運動的時間、讀書的時間、玩樂的時間，我想知道我一週都做了些什麼，時間究竟浪費在哪？再去慢慢調整。

八月份好多身邊的朋友換工作了，自己多少也有那些念頭浮現，好像是時候該去外面闖一闖了（？）但是我也告訴自己不是現在，不要受其他外物影響。這個八月是我第一個非暑假的八月，今年畢業生陸續開始找工作後，開始正式意識到自己已經不是學生樣了，我更在意的是如何讓自己更有能力，更像個成熟大人。

省思

反思了一下自己，覺得自己還不夠珍惜現在擁有的，我對於吹長笛很感興趣其實也是最一開始老師借我長笛，如果老師只借我一下就要求我自己買一把，我覺得我可能不會練習得這麼勤奮，你會有種「老師如果把長笛收回去要我自己買一把的話怎麼辦 QQ？我要珍惜每一次練習可以用長笛的時候」。這個心態換到任何上面都是，健身房之前每天開的時候我也沒有去很勤，也不是很認真練，結果疫情下來不能去就是三個月，不珍惜可以訓練的時候，沒了才再後悔，都是無用的，而且自己想想我先天條件已經比別人好很多了，有什麼資格抱怨然後不努力呢？而工作上也是，工作沒有很忙碌的話，我為什麼不好好珍惜寫專案以外的時間，把時間拿來看別人的 Code 學點新東西什麼的都好。總而言之呢，我要珍惜現在且更加努力利用現有的資源，這是我八月的心情感觸。
